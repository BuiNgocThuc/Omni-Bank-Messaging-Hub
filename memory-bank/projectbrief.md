# Project Brief

Omni-Bank Messaging Hub is a microservices-based, event-driven system for processing Foreign Exchange (FX) transactions. 

## Core Requirements & Goals
- Support buying and selling foreign currency.
- Manage account balance holds (Hold Balance).
- Ensure Double Entry Accounting for ledger consistency.
- Utilize Message Queue for asynchronous processing.
- Maintain idempotency for transaction requests.
